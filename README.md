# PDF Master v.3

Ferramenta profissional local e segura para manipulação de PDFs diretamente no navegador. 100% privado, nenhum arquivo sai do seu computador.

## Funcionalidades Principais
- **Unificar PDFs**: Combine vários arquivos em um único documento.
- **Separar PDFs (Avançado)**:
  - **Por Intervalos**: Extraia múltiplos intervalos de páginas (ex: 1-10, 20, 30-35) de cada arquivo. Cada intervalo se torna um novo arquivo.
  - **Páginas Individuais**: Transforme cada página em um arquivo PDF separado.
  - Receba múltiplos resultados organizados em um pacote ZIP.
- **Seleção Flexível**: Suporte a strings de intervalo complexas e intuitivas.
- **Remover Metadados**: Limpe informações sensíveis e histórico do documento.
- **Compressão de Alta Performance**:
  - **Qualidade Livre**: Ajuste manual de 10% a 100%.
  - **Tamanho Máximo**: Heurística inteligente que prioriza a legibilidade.
  - **Modo Tons de Cinza**: Converta para P&B para manter maior nitidez em tamanhos reduzidos.
- **Personalização de Nome**: Defina exatamente o nome do arquivo de saída.

## Variantes de Layout
Esta versão inclui duas opções de interface para sua preferência:
1.  **UnificaPDF.html (Padrão)**: Design v3 moderno, limpo e focado em UX. Suporte total a modo claro/escuro.
2.  **UnificaPDF_Obsidian.html**: Visual sofisticado com "glassmorphism", agora totalmente em português e com suporte a modo claro/escuro.
